How it works

  • Data Pre-Processing

    The heart disease data is pre-processed after a collection of various records. To work with categorical variables, certain attributes are broken down into dummy columns using 1's and 0's. The get dummies method is used to split categorical questions into yes or no variables for each possible answer.

  • Feature Selection

    All attributes of the data set contain vital clinical information and are all relative. Out of all the attributes, age and the sex are used to identify the demographic information of the subject. The data takes the form of a questionnaire and so each attribute answer for each question plays a role and determines the outcome.

  • Classification and Prediction

    The performance of each classifier is evaluated for accuracy and loss. To evaluate the heart disease risk prediction, these are the following tools and classifiers:
    Random Forest Classifier
    Adapting Boosting Classifier
    Keras Sequential Model

  • Performance Evaluation

    Random Forest Classifier: 90.2 %
    Adapting Boosting Classifier: 91%
    Keras Sequential Model: 91.6%
